Format 1 — Firebase RTDB Export

Upload a raw Firebase Realtime Database export. ELO values are read from the ELO key. Stats (total_games, total_races, total_wins) are computed automatically from the act-history key if present. 

{
  "ELO": {
    "Mario": 1523,
    "Luigi": 1478
  },
  "act-history": { ... }
}

Format 2 — Player Array

Upload a JSON array of player objects. Only name and elo are required. 

[
  {
    "name": "Mario",
    "elo": 1523.4,
    "total_wins": 42,
    "shirt_swap_count": 38,
    "total_games": 20,
    "total_races": 320,
    "head_to_head": {
      "Luigi": { "wins": 12, "losses": 8 }
    }
  }
]

Players are matched by name. Existing players are updated; new names are created.
